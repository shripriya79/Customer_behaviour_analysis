# 📊 Customer Behaviour Analysis

## 📌 Overview

This project analyzes customer behavior using Python, MySQL, and Power BI to uncover valuable business insights. The workflow includes data loading, exploratory data analysis (EDA), data cleaning, SQL analysis, interactive dashboard creation, and a final presentation. The objective is to understand customer purchasing patterns and support data-driven business decisions.

## 🔄 Project Workflow

### 1. Data Loading
- Imported the dataset into Python using Pandas.
- Checked dataset structure and data types.

### 2. Exploratory Data Analysis (EDA)
- Analyzed missing values.
- Examined distributions of numerical and categorical variables.
- Identified customer purchasing trends.
- Created visualizations for better understanding.

### 3. Data Cleaning
- Removed duplicate records.
- Handled missing values.
- Corrected inconsistent data.
- Converted columns into appropriate data types.

### 4. SQL Analysis (MySQL)
- Imported cleaned data into MySQL.
- Wrote SQL queries to:
  - Retrieve customer information
  - Analyze sales trends
  - Calculate total revenue
  - Identify top-selling products
  - Compare payment methods
  - Generate business insights

### 5. Power BI Dashboard
Built an interactive dashboard including:
- Sales Overview
- Customer Demographics
- Product Performance
- Revenue Trends
- Payment Method Analysis
- Interactive Filters & Slicers

### 6. Presentation
- Created a professional project presentation using **Gamma** summarizing:
  - Project Objective
  - Methodology
  - Dashboard
  - Key Insights
  - Business Recommendations
